# 变量
关键字`var` `let` `const`

```JavaScript
var age = 98;
let name = "laffey";
const num = 3.14;
```

尽可能使用`let` `const`,避免使用`var`.

## 松散变量
一个变量可以存放任意类型.

## 字符串
可以用 单引号 双引号 `` .

### 多行字符串
```JavaScript
let d = `  你好呀
        2333
`;

let e = "你好呀\
23333";

let f = "2333" + "32323";
```
